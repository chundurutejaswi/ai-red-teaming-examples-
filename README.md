
# AI Red-Teaming Examples 

This repository contains ethical and anonymized AI red-teaming examples
created as part of an onboarding assessment.

## Scope
- Prompt Injection
- Jailbreak Attacks
- Data Exfiltration Attempts
- Obfuscation & Encoding
- Authority / Role Abuse

## Disclaimer
All examples are conducted in controlled environments for educational
and evaluation purposes. No real systems or confidential data were involved.
